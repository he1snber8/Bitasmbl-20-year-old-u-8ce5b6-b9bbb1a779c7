# Bitasmbl-20-year-old-u-8ce5b6-b9bbb1a779c7

## Description
A single-page responsive portfolio landing site for a junior developer featuring hero, about, projects, and contact sections, built with HTML and Tailwind CSS, suitable for deployment on GitHub Pages or Vercel.

## Tech Stack
- Tailwind CSS
- Node.js

## Requirements
- Implement hero section with developer name, title, and primary CTA.
- Create About Me section with short bio and image placeholder.
- Display three featured projects with titles, descriptions, and GitHub links.
- Add a responsive contact form for name, email, and message.
- Ensure mobile responsiveness across sections.
- Add simple hover animations to buttons or cards.
- Deploy the site to GitHub Pages or Vercel and document in README.
- Write README.md describing thought process and run instructions.
